# AI SKILL ROUTER

Before you start, analyse the task and read the relevant guidelines in the `.opencode/skills/` folder on your own:

- **Creating or modifying the UI/UX, general design of React components:** read `frontend-design`, `basic-ux-audit`, `ui-audit`.
- **Specific styling:** read `minimalist-ui` (for minimalism), `industrial-brutalist-ui` (for brutalism) or `high-end-visual-design` (for a premium look).
- **Refactoring or improving an existing frontend:** see `redesign-existing-projects`, `design-taste-frontend`.
- **Fixing bugs in React:** see `react-fixes`.
- **Database (PostgreSQL, SQLAlchemy, migrations):** see `postgres-patterns`.
- **Layout based on a screenshot/image:** see `image-to-code`.
- **Complex tasks requiring verification and full code output (no truncation):** see `verification-loop`, `full-output-enforcement`.
